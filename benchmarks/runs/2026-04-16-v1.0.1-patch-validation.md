## Patch Goal

`v1.0.1` exists to close the global dry-run safety regression from `v1.0.0`.

Required fix:

- `bash installers/install-global.sh --dry-run /missing/path` must not create `/missing/path`
- global preflight must still classify the missing target honestly
- real install on that same missing path must still work

- repo doctor

- missing-target global dry-run regression
- missing-target first install path
- no-`rg` fallback in the shipped verification script
- second Linux VPS verification
