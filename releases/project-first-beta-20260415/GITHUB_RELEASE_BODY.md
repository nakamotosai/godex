# godex Project-First Beta

`godex` is an open-source upgrade kit for Codex that turns stronger task mirroring, verification discipline, and lower-noise closeouts into an installable product.

## In This Beta

- project install with backup, restore, doctor, benchmark, and example pack
- global install for one Codex home with backup, restore, doctor, benchmark, and example pack
- clean-surface install proof on isolated repo and Codex-home targets
- stronger review-output layout rules for long findings

## Install Modes

- quick paste: `prompts/bootstrap/GODEX_BOOTSTRAP.md`
- project install: `docs/PROJECT_INSTALL.md`
- global install: `docs/GLOBAL_INSTALL.md`

## Known Boundary

- complex existing custom global policies may still require manual review
- proof is broader than maintainer-only dogfood, but not yet broad community validation
- this is a public beta, not `v1`
