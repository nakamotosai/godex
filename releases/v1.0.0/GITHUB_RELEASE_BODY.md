# `godex` v1.0.0

`godex` upgrades vanilla Codex into a clearer, lower-noise, verification-first engineering agent without turning the product into a giant prompt dump.

## What `v1.0.0` Includes

- quick paste bootstrap
- project install
- global install
- backup, restore, doctor, and benchmark for both durable install modes
- dry-run preflight before risky mutation
- safe `manual_review` stop for complex existing `AGENTS.md` policies
- troubleshooting, upgrade, support matrix, and GitHub support surfaces
- CI release guard and validation records

## Start Here

- quick paste: `prompts/bootstrap/GODEX_BOOTSTRAP.md`
- project install: `docs/PROJECT_INSTALL.md`
- global install: `docs/GLOBAL_INSTALL.md`
- troubleshooting: `docs/TROUBLESHOOTING.md`
- upgrade: `docs/UPGRADE.md`

## Known Boundary

- complex global policy stacks stop for manual merge instead of auto-merge
- support target is Codex CLI on Unix-like shells
- this is a formal `v1`, but not a claim of universal runtime support or broad community-scale validation
