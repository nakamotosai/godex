# `godex` v1.0.0 Release Pack

This directory contains the release assets for the first formal `godex` release.

## Included Assets

- `RELEASE_NOTES.md`
- `GITHUB_RELEASE_BODY.md`
- `X_ANNOUNCEMENT.md`
- `KNOWN_BOUNDARIES.md`
- `POST_TAG_SMOKE.md`
- `CHECKLIST.md`

## Release Goal

`v1.0.0` is the first release that should be supportable for strangers on GitHub, not only for the maintainer's own dogfood workflow.
