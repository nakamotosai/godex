# examples

This directory holds public showcase material for `godex`.

## Shipped Example Packs

- `project-install-beta/`
- `global-install-beta/`

## Why This Matters

Examples are the fastest way to make the product feel real to a new visitor.

They also keep the repo honest by showing the exact behavior change being claimed.

- `global-install-beta/` is the global-surface example pack
